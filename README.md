# OpenGL---CuboRubik
Proyecto de un Cubo Rubik interactivo en 3D desarrollado con **C++** y **OpenGL**.  
El programa permite visualizar y manipular un cubo rubik mediante controles de teclado y renderizado en tiempo real.

---

## 📸 Características

- Renderizado 3D usando OpenGL
- Interacción en tiempo real
- Rotación de caras del cubo
- Animaciones suaves
- Cámara manipulable
- Implementado completamente en C++

---

## 🛠 Tecnologías utilizadas

- C++
- OpenGL
- GLUT / FreeGLUT
- GLEW (opcional)
- GLFW (si aplica)

---

## 🚀 Instalación

### Requisitos

Antes de ejecutar el proyecto asegúrate de tener instalado:

- Compilador C++
- OpenGL
- FreeGLUT o GLFW
- CMake (opcional)

---
